# Reset Database

Reset and reseed the development database.

## Instructions

1. Confirm this is a development environment (check NODE_ENV)
2. Run `npm run db:reset` to reset the database
3. Run `npm run db:seed` to seed sample data
4. Verify the database is working by checking `npm run db:studio` or running the app
5. Report completion status

WARNING: This destroys all existing data. Only use in development.
