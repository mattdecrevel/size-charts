# Build and Verify

Run a full production build and verify everything works.

## Instructions

1. Run `npm run lint` to check for linting errors
2. Run `npm run build` to create a production build
3. If build fails, analyze errors and fix them
4. After successful build, run `npm run test:run` to verify tests pass
5. Report build status, any warnings, and bundle size if available

Fix any errors encountered during the build process.
