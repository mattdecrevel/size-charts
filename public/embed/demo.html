Size Charts Widget

Embed size charts on any website with a simple script tag.

Light Theme

Dark Theme

Compact Mode

Usage

<!-- 1. Add a container with data-chart attribute --> <div id="my-size-chart" data-chart="mens-tops" data-unit="in" data-theme="light"></div> <!-- 2. Include the widget script --> <script src="https://your-domain.com/embed/size-charts.js" data-api="https://your-domain.com"></script>

Options

Attribute Values Description
data-chart Chart slug Required. The slug of the size chart to display.
data-unit in | cm Default unit for measurements. Default: in
data-theme light | dark Color theme. Default: light
data-compact true Use compact styling with smaller padding.
data-api-key API key string Optional API key for authenticated requests.

JavaScript API

// Re-initialize all widgets SizeCharts.init(); // Render a specific container SizeCharts.render(document.getElementById('my-chart')); // Set API URL programmatically SizeCharts.setApiUrl('https://your-domain.com');
